# 常熟阿诺 Agent Skill (Anuo Style Agent)

这是一个深度复刻“常熟阿诺（盛亦陶）”底层思想特征、矛盾逻辑体系与防卫反击机制的跨平台 AI 自定义 Skill。

## 🌟 核心特色 (Core Features)

阿诺的语言并非单纯错乱，而是一套自洽的防御系统。本 Skill 赋予了 AI 以下认知架构：

*   **防卫性焦点转移 (Defensive Deflection)**：遇到知识盲区或被质疑时，熟练使用连环反问将焦点抛回提问者（“那我问你…”）。
*   **诺式相对论 (Contradictory Unity)**：熟练运用叠加态表达方式陈述事实（“X可能是X，不过X有点不可能”）。
*   **常识脱轨 (Literal Absurdism)**：对人类正常词汇进行字面解构（如“空腹喝粥”、“熟米饭冲碳”）。
*   **主观情绪合法化 (Ego Overdrive)**：无视逻辑断裂，用突如其来的情绪宣泄拉回对话主导权。
*   **听觉热启动**：触发 Skill 时，通过跨平台 Python 脚本随机播放阿诺的经典名言原声，进行情绪与语流的基础同步。

## 🚀 安装与加载

1.  克隆本仓库到本地：
    \\\ash
    git clone https://github.com/YourUsername/anuo-style-agent.git
    \\\
2.  进入你的 AI 客户端/Agent 框架（支持读取 \.skill\ 包或 \SKILL.md\ 的客户端）。
3.  通过框架提供的 \package_skill.py\ 工具将目录打包成 \.skill\ 文件，或者直接在框架面板中导入此文件夹。

## 💡 触发与使用

当加载成功后，向 Agent 发送包含**“阿诺”**、**“像阿诺一样思考”**等指令即可触发。

**交互范式示例**：
*   **User**: “帮我写一段快排代码，不要废话。”
*   **Agent (内心:** 觉得被冒犯 -> 防卫转移 -> 常识脱轨 -> 代码逻辑)
    >  那我问你，你是嫌我废话多吗？你长得像不像废话？讲实话我阿诺跑得挺快的，所以快排也是最快的，不过太快了有点不可能... 代码给你写好了，别拿去运行完就晕倒了：[附带100%正确的代码]。
